Collaboration tools

We currently use the following tools to collaborate:

  • This lab wiki for information, lab policies and how-tos
  • SURF Research Drive, for documentation that should not be made public and research data
  • Slack for group and one-on-one chats
  • GitHub for sharing code
  • Open Science Framework for sharing research data and documentation for each research project
  • Zoom, Teams, Padlet, Miro, Menti and others during (online) meetings
